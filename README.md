# Simple Login Dashboard API

A responsive login-based dashboard web application built using HTML, CSS, and JavaScript.  
It includes login validation, dashboard UI, logout functionality, and API data fetching.

## Features
- Login form with validation
- Dashboard with welcome message
- Logout button
- localStorage session handling
- Fetch data from public APIs
- Display news, weather, and location
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
- Public APIs
